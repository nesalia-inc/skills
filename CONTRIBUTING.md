# Contributing to Marty Skills

We welcome contributions from team members at Nesalia Inc. and DeesseJS.

## How to Contribute

1. Fork the repository
2. Create a feature branch from `canary`
3. Make your changes following the skill conventions
4. Submit a pull request to `canary`

## Conventions

- All skills must have YAML frontmatter with `name`, `description`, `context`, `agent`, `permissionMode`, and `allowed-tools`
- Use non-interactive commands only (no editors)
- Include Co-Authored-By in all commits
- Follow conventional commits format

## Questions
